# Catetan javascript

## Variable

- `let`  = Bisa di assign ulang
- `const` = Gabisa di assign ulang


## Tipe data

- **string = Teks**
- **number = Angka**
- **undefined = Value kosong**
- **null = Value sengaja kosong**


## Operator 

| TAMBAH | KURANG | KALI | BAGI |
|--------|--------|------|------|
|    +   |    -   |   *  |   /  |
|--------|--------|------|------|

- == vs ===


## Operator perbandingan 

- == = **Sama dengan**
- === = **Sama dengan / Tipe data sama**
- != = **Tidak sama dengan**
- !== = **Tidak sama dengan / Tipe data beda**
- < = **Lebih kecil**
- > = **Lebih besar**
- <= = **Lebih kecil atau sama**
- >= = **Lebih besar / Sama**



## Percabangan


### if else 

`const usia = 20`

```javascript
if (usia >= 17){
  console.log("Sudah Dewasa")
}else{
  console.log("Masih Kecil")
}
```

### if..else if..else

`const usia = 20`

```javascript
if (usia >= 18){
  console.log("Boleh bikin ktp")
}else if (usia === 17){
  console.log("Tunggu umur 18")
}else{
  console.log("Belum boleh bikin ktp")
}
```
